# pyMIND

This is the udpated web-based pyMIND application.

To run the backend, follow these instructions:
- uvicorn backend.main:app --reload
- http://127.0.0.1:8000/api/graphs/eeg/live
- http://127.0.0.1:8000/api/graphs/eeg/trend
- http://127.0.0.1:8000/api/vitals/live
- http://127.0.0.1:8000/api/vitals/trend
